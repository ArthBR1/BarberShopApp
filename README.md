# Barber Shop App
APP Full Stack de agendamento de barbearia utilizando Java e Angular

## Tecnologias utilizadas
### Para a interface de usuário:
- HTML
- SCSS
- Javascript
- Typescript

### Para a API:
- Java
- SQL
- Gradle
- Spring
